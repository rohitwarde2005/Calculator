# 🧮 Calculator Web App

A simple and stylish calculator built using **HTML**, **CSS**, and **JavaScript**, featuring:

- Basic arithmetic operations
- Responsive design
- Light and Dark Mode toggle (☀️ 🌙)
- Local storage theme memory
